Australian Open Tennis Matches Statistics

Some of the most important stats related to tennis based on Australian open are visualized below. Click on each stat for winner and loser statistics over years. There is also an additional filter which can be applied through the drop down below to look at the stats of very popular players in Australian Open.

Story

These visualizations briefly show which element should be focused by any tennis player should focus on to improve his/her game

The first visualization which is a bubble graph contains aggregate statistics contributing to the winner in the tournament. Bubble chart is a interactive graph and when clicked on any circle displays a bar graph

The bar graph contains data corresponding to winner and loser statistics over the years for the particular selected feature

Additionally, a filter can be applied through the dropdown using which most popular players stats in Australian Open tournament can be visualized
